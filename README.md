# GriffNode for BigCommerce

Accept Bitcoin, Ethereum, Litecoin, Dogecoin and more in your **BigCommerce** store — settled straight to
your own wallet, no chargebacks, no KYC. Powered by [GriffNode](https://griffnode.com).

> **Status:** the app is live on the BigCommerce marketplace; its source is being imported into this repo,
> after which tagged GitHub Releases will publish new versions (see `.github/workflows/release.yml`).

- **Marketplace:** https://www.bigcommerce.com/apps
- **API reference:** https://docs.griffnode.com/reference
- **Get an API key:** https://griffnode.com
